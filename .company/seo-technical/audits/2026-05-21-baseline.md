## 🔴 致命的問題

### Issue #1: shop.html が SPA構造で全店舗共通の静的HTML

**現状**:
- ソースHTML title: `店舗情報 - KaigaiQ`（全店舗共通）
- description, canonical, OG, Twitter Card すべて未設定
- JS実行後に shop.js が `pageTitle` 要素のtitleを書き換える（Googlebotがレンダリングしないと見えない）

**影響**:
- 個別店舗（35店）の検索クエリでヒットしない
- SNSシェア時に全店「店舗情報 - KaigaiQ」と表示
- canonicalがないため `?id=A` `?id=B` がそれぞれ重複コンテンツと判定されるリスク

**優先度**: 最高

---

### Issue #2: news.html の meta description が空

**現状**:
```html
<meta name="description" content="">
```

**影響**:
- 検索結果スニペットがGoogleの自動生成に委ねられる
- CTR大幅低下

**優先度**: 高

---

### Issue #3: 画像 alt属性 が大半未設定

**現状**:
- index.html: img要素は CSS background-image で背景画像化されているものが多く、明示的なalt付きimgは少数
- shop.html: 1個のimg、alt未設定

**影響**:
- 画像検索流入の取りこぼし
- アクセシビリティスコア低下

**優先度**: 中

---

### Issue #4: JSON-LD構造化データの不足

**現状**: TOP/shop/news に各1個のJSON-LDが存在するが、業態に必要なSchemaが欠落

**欠落しているSchema**:
- `LocalBusiness` / `NightClub` / `BarOrPub`（個別店舗ページ）
- `BreadcrumbList`（全ページ）
- `JobPosting`（個別店舗の求人情報）
- `Article` / `NewsArticle`（ニュース個別記事）
- `FAQPage`（FAQセクション）
- `WebSite` SearchAction（TOPには既にあるか要確認）

**影響**:
- Google for Jobs に求人が掲載されない
- リッチリザルト（パンくず、FAQ、レーティング）が表示されない
- ローカルパック表示の機会喪失

**優先度**: 高

---

## 🟡 中程度の問題

### Issue #5: news.html の OG/Twitter Card が部分設定

OG: 4個（不完全）、Twitter: 1個（最小限）。`og:image` / `twitter:image` の有無を要確認。

### Issue #6: shop.html の canonical 動的設定がない

`?id=Rashell` を canonical で `https://kaigaiq.com/shop.html?id=Rashell` として正規化する必要がある。

### Issue #7: hreflang 未設定

KaigaiQは日本語のみだが、`<html lang="ja">` は設定済。海外展開を視野に入れる場合は将来対応。

### Issue #8: Core Web Vitals 未計測

PageSpeed Insightsで LCP / CLS / INP を実測してベースラインを残す必要がある。
